Detailed Description

Sistema de confort térmico inteligente para aula (documentado para Doxygen).

Este archivo contiene el firmware principal para un sistema embebido basado en Arduino que monitorea temperatura, humedad y luz para mantener condiciones de confort térmico en un aula de dos estudiantes. El sistema emplea una máquina de estados (StateMachineLib), tareas asíncronas (AsyncTaskLib), lectura de RFID (MFRC522), DHT11, NTC, LDR, y controla actuadores como relé, servo y buzzer.

◆ doMonitorStep()

void doMonitorStep ( )

Ejecuta un paso de monitoreo: lee sensores, actualiza medias y decide señales.

Lecturas:

  • NTC -> temperatura_actual (promediado)
  • DHT -> humedad_actual (promediado)
  • LDR -> luz_actual (ADC)

Calcula pmv_actual a través de pmv_from_table. Actualiza LCD con datos. Control de transiciones: alarma por temperatura, señales PMV alto/bajo.
